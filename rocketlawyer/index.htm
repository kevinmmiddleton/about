Rocket Lawyer's Mission

Project's Contribution

The Challenge

 

Users

  • Existing: Individual consumer and company benefit legal services.

  • New: Building for partners, starting with Covéa, an insurance holding company with 3 distinct brands serving 11.5 million members.

     

Pain Points Addressed

Providing partner customers with digital legal documents, eSigning, and 'Ask A Lawyer' capabilities.

Solution

Overview

Developed a comprehensive solution integrating Rocket Lawyer's services with partners, focusing on seamless authentication, co-branding, customized pricing, and exclusive partner content.

 

 

Features

  • Partner Authentication: Implemented using OpenID Connect for robust, scalable partner integration.
  • Custom Connector: Partner-specific integrations.
  • Partner Config API: Customizable settings and frontend brandability for partner-specific requirements.
  • Reporting Capabilities: Enhanced existing events structure to provide detailed, partner-specific reporting.
  • Co-Branding and Partner-Specific Pricing: Customizable for each partner.
  • Partner-Exclusive Content: Tailored legal solutions and resources.

Technical Challenges

 

Authentication and Connector

 

Challenge: Developing a robust, scalable solution for partner authentication.

 

Solution: Incorporated OpenID Connect with an optional custom connector, ensuring scalability and flexibility.

Technical Challenges

 

Partner Config API

 

Challenge: Creating a system for flexible, multi-brand controls and settings.

 

Solution: Created the Partner Config API for customizable partner settings, frontend brandability, and differentiating Rocket Lawyer's country websites as unique brands.

Technical Challenges

 

Reporting Capabilities

 

Challenge: Track and report partner usage, including usage-based billing.

 

Solution: Enhanced the in-house user events framework, enabling detailed usage reporting and billing model support.

Reflection and Key Decisions

 

Prioritization

Delivering a slim but impactful Minimum Lovable Product (MLP), prioritizing features that provided the most value to partners.

 

Building for Scale
